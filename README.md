# adq-maha-test

### Used tech stack:
Java 11, Kotlin, gradle, Spring MVC, Spring Data JPA, Spring Testing, H2, Junit5, Swagger

### How to build:
`./gradlew build`

### How to run:
`java -jar ./build/libs/test-0.0.1-SNAPSHOT.jar`

### Testing:
Swagger UI is available at `http://localhost:8080/swagger-ui.html`.
